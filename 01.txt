1)O que é o insumo, ou entrada, em um sistema de informação?
R:O insumo ou entrada é tudo oque entra no sistema para ser processado, o sistema só utiliza o necessário.
2)Onde podemos coletar dados para sistemas? Explique e cite exemplos.
R:Os dados são coletados com os usuários do sistema, com as pessoas que pediram a construção do sistema, com suas opiniões e sugestões.
3)Explique o que é um desenvolvedor backend, frontend, e full stack;
R: O desenvolvedor backend é aquele que fica com a parte da programação do sistema, com toda a parte burrocrática e inplementação das ideias.
   o desenvolvedor frontend é aquele que fica com toda a parte web do sistema, ele faz o sistema ficar bonito para o usuário, e tem como papel principal  deixar o sistema com uma boa usabilidade para o cliente.
   O desenvolvedor full stack é aquele que sabe fazer o backend e o frontend.
4)Qual a diferença entre frontend e designer?
R: O frontend é quem desenvolve a parte web do sistema.
   O designer cria as imagens para serem implementadas no sistema,a parte gráfica, deixando mais bonito e chamativo.
5)Quem é o responsável por manter um servidor de banco de dados e quais são suas obrigações?
R:O responsável é o DBA, que controla toda a parte do banco de dados, armazenando la os dados e garantido a segurança do armazenamento.
6)Que dados uma nota fiscal eletrônica possui? Cite todos e o objetivo de ter esses dados.
R: Ela registra eletrônicamente as operações tributárias dos empreendimentos,tem uma válidade jurídica que é assegurada pela assinatura digital da empresa emitente e pela autorização prévia de uso pela administração tributária do estado responsável pelo recolhimento do tributo,a troca da antiga nota fiscal pela nova nota fiscal eletrônica proporcionou inúmeras vantagens,como(a exclusão das notas de papel, a facilitando o dia a dia do trabalhador).Nela são armazenados todas as informações do cliente e do emitente, esses dados são armazenados online, no site da receita, ficando diponivel para pesquisas por ate 180 dias.
Ela reduz o impacto ambiental por nao ser mais de papel,reduz o tempo de entrega das mercadorias, diminui os custos com a impressão, incentiva o comércio eletrônico, aumenta a segurança e credibilidade, amplia o controle fiscal.
7)Quais os benefícios em versionar código fonte em ambientes como o GitHub?
R:Que no GitHub varias pessoa podem ver seus códigos, identificar possiveis erros e avisar o criador para corrigir o problema, assim todos se ajudam, porque ali podem ser compartilhados códigos fonte.
